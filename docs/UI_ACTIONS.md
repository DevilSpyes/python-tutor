# UI Actions & Shortcuts

This document outlines the available actions in the CyberTutor Hacker HUD interface.

## General Navigation

| Action | Description | Shortcut |
| :--- | :--- | :--- |
| **Toggle Sidebar** | Opens/Closes the left navigation panel. | Mobile: Hamburger Menu |
| **Toggle AI Panel** | Opens/Closes the right AI assistant panel. | Mobile: Robot Icon |

## Left Panel (Modules)

| Action | Description |
| :--- | :--- |
| **Search** | Filter modules and lessons by title. |
| **Filter Tags** | Click tags (ALL, BASIC, ADV) to filter by category. |
| **Expand/Collapse** | Click module headers to toggle visibility of lessons. |
| **Select Lesson** | Click a lesson to load it into the editor. |

## Code Editor & Typing Simulator

| Control | Icon | Description |
| :--- | :--- | :--- |
| **Play/Pause** | ▶ / ⏸ | Starts or pauses the "hacker typing" effect. |
| **Skip** | ⏭ | Instantly finishes typing and shows full code. |
| **Replay** | ↺ | Clears editor and restarts the typing animation. |
| **Speed** | Slider | Adjust typing speed from 0.25x (slow) to 2x (fast). |
| **Run Code** | ▶ RUN | Executes the current code in the browser (Pyodide). |

## AI Assistant

| Action | Description |
| :--- | :--- |
| **Send Message** | Type query and press Enter or click ▶. |
| **Settings** | Click ⚙️ to configure API keys or Local LLM mode. |
| **Clear History** | Available in Settings to reset the conversation. |

## Keyboard Shortcuts (Planned)

- `Ctrl + Enter`: Run Code
- `Ctrl + /`: Focus AI Chat
